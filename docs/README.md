## ğŸŒ Overview

**@woby/wui** is a comprehensive, Material-inspired UI component system for Woby.  
It provides:

- A wide selection of ready-to-use components
- Full support for reactivity using Woby signals
- Customizable styling via `cls`
- Accessible and consistent UI patterns

The library is designed to be lightweight, extensible, and developer-friendly.
